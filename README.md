
# Хронологічний трекер читання Біблії

Простий веб-додаток для щоденного хронологічного читання Біблії. Прогрес і нотатки зберігаються у `localStorage`.

Файли:
- `index.html` — головна сторінка (Погляд на сьогодні + список усіх днів)
- `archive.html` — архів нотаток
- `style.css` — стилі (mobile-first, оптимізовано для 390px)
- `plan.js` — масив `biblePlan` з днями та розділами
- `script.js` — логіка додатка

Запуск локально:

```bash
# з папки проекту
python3 -m http.server 8000
# відкрити http://localhost:8000 у браузері
```
